# 咕嚕咕嚕傳統小吃 — 點菜系統使用說明

## 檔案說明
| 檔案 | 用途 |
|------|------|
| `咕嚕咕嚕_QR列印頁.html` | 店家列印用：7張QR Code（桌1~6 + 外帶） |
| `咕嚕咕嚕_點菜頁.html` | 客人掃QR後開啟點菜 |
| `咕嚕咕嚕_廚房出單.html` | 廚房螢幕即時接單 |

## 使用流程
1. 用電腦開啟「QR列印頁」→ 列印 → 裁剪放到各桌
2. 廚房螢幕開啟「廚房出單頁」→ 保持開著
3. 客人掃QR → 手機開啟「點菜頁」→ 點餐送出
4. 廚房頁面自動彈出新訂單通知

## 注意
- 三個頁面需在同一台電腦/同一瀏覽器開啟才能用localStorage通訊
- 實際部署建議放到同一網域的網頁伺服器
